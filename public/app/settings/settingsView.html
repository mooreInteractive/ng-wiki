Wiki Settings

Below you can change some of the global properties of this ng-Wiki installation.

Wiki Name

The main title of the wiki. Shows up in the Main Header on every page.
Tagline

A phrase or slogan to sit just below the wiki title in the main header on every page.
Welcome text

A welcome message at the top of the home page.
Message of The Day

A message of the day. Sits just below the Welcome message on the home page.
Home Page HTML

Any additional HTML you'd like to include on the homepage. If left empty, a Category listing will be shown.
